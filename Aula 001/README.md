# Aula 001 - Fundamentos, Contêineres e Setup do Ambiente WSL/Linux

## Visão Geral
**Módulo:** I - Fundamentos e Setup  
**Duração:** 3h30min  
**Foco:** Instalação WSL2 e comandos Linux essenciais
![InfoGrafico Aula 1](InfoAula001.png)

## Objetivos da Aula
- Configurar ambiente WSL2 no Windows
- Dominar comandos Linux fundamentais
- Compreender conceitos de virtualização e containerização
- Estabelecer base para desenvolvimento em ambiente Linux

## Conteúdo Programático
- Conceitos de virtualização e containerização
- Instalação e configuração do WSL2
- Comandos Linux fundamentais (navegação, manipulação de arquivos)
- Permissões e administração básica

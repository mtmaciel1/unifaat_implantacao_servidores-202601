1.  Responda às questões teóricas com base no conteúdo da aula.

## Questão 1: Arquitetura Docker (Teórica)
Um desenvolvedor executa o comando `docker run [imagem]` no seu terminal WSL. Descreva, em uma sequência de passos, o fluxo de comunicação que ocorre, mencionando:

a) O papel do Docker Client (o terminal).

b) O papel do Docker Daemon (o servidor que executa a ação).

c) O papel do Docker Registry (no caso, o Docker Hub, se a imagem não estiver local).
